Factors Affecting Respiratory Modulations

Summary

In this article the toolbox (v.3) was used to assess a range of technical and physiological factors affecting respiratory modulations of the ECG and PPG. Respiratory modulations were extracted from the signals in the Vortal dataset (young and elderly subjects) using a wide range of techniques. The correlations of each respiratory modulation with a reference respiratory signal were calculated. This allowed us to investigate the effect of several technical factors (including site of PPG measurement, type of recording equipment, input signal (ECG or PPG) and sampling frequency), and physiological factors (including age, gender, heart rate and respiratory rate) on the respiratory modulations.

Replicating this study

This study can be replicated as follows:
  1. Download the data file corresponding to both young and elderly subjects in the Vortal dataset.
  2. Use Version 3 of the toolbox of algorithms.
  3. Downsample signals using the following Matlab script: run_vortal_sampling_freqs
  4. Perform respiratory signal extraction and correlation calculations using the following Matlab command: RRest('vortal_factors')
  5. Perform statistical analysis using the following Matlab script: run_vortal_factors
