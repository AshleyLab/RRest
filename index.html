Welcome to the "Respiratory Rate Estimation" project page.

The aim of the "Respiratory Rate Estimation" project is to provide open-source resources to assist researchers with the development of algorithms for estimation of respiratory rate from commonly measured physiological signals. The project is introduced as follows:

  1. Background: An overview of the rationale for estimating respiratory rate from physiological signals.
  2. Algorithms: A summary of the toolbox of algorithms to aid research into estimation of respiratory rate, and a link to the algorithm documentation.
  3. Datasets: Links to datasets which can be used with the toolbox to evaluate respiratory rate algorithms.
  4. Related Publications: A selection of publications which used the code and/or datasets referred to in this project.
  5. Contributions: Details of how to contribute to this ever-growing resource.
